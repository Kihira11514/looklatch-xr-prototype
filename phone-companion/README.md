# Phone Companion Stub

This module represents the trusted Android phone side of LookLatch XR.

Current scope:

- `ProximityProvider` interface for proximity validation.
- `FakeProximityProvider` for local demos.
- `UwbProximityProvider` placeholder for future Android UWB/Ranging integration.
- `BleSessionProvider` placeholder for future BLE out-of-band session setup.

The phone companion never stores real vehicle keys or production credentials in this prototype.
