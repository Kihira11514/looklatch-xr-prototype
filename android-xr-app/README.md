# Android XR App Stub

This module is a Kotlin/Compose placeholder for the LookLatch XR glasses experience.

Current scope:

- No real Jetpack XR SDK integration.
- No real camera, gaze, vehicle, lock, or digital-key integration.
- Shows the safety state machine screens used by the concept demo.

The app demonstrates that gaze can only move the flow into an armed state. Unlock is simulated and requires a proximity signal plus explicit physical confirmation.
